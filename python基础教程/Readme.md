阶段三：掌握三大Python后端框架，解决一切前后端Web开发问题

阶段四：掌握三大Python后端框架，解决一切前后端Web开发问题

阶段五：Web框架之Tornado

阶段六：docker容器及服务发现

阶段八：成为Python数据挖掘分析师，进入人工智能领域，成为IT市场最前沿人才
